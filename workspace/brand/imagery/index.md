## Tagging Convention

Use these categories when adding new images:

- **Type:** product, photography, abstract, icon, data-viz
- **Subject:** portal, api, network, server, team, office, data-center
- **Tone:** professional, technical, warm, bold, minimal
- **Color:** dark, light, green, citron, neutral
- **Persona:** cio, developer, devops, marketing, executive

## Image Requirements

- PNG or high-quality JPG
- Minimum 2400px on longest edge
- No baked-in text or logos
- Clean edges for bleed/crop
- Transparent background (PNG) preferred when possible

## Adding Images

Via Slack: Upload image to creative ops channel with `@adgen save this as [type], tags: [tags]`
Via script: Run `generate_samples.py` for programmatic generation
